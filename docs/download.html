✨ 업데이트는 자동입니다

설치 후에는 PowerPoint를 켤 때마다 애드인이 최신 버전을 스스로 확인합니다. 새 버전이 있으면 패널 상단에 "업데이트 준비됨" 배너가 뜨고, 클릭하면 PowerPoint가 재시작되며 자동 적용됩니다. 다시 다운로드할 필요 없습니다.

1 최초 설치 (1회만)

  1. 위 다운로드 버튼으로 zip을 받습니다.
  2. zip을 압축 해제합니다 (아무 폴더나 OK — 설치 시 고정 위치로 복사됨).
  3. install.bat을 마우스 우클릭 → 관리자 권한으로 실행합니다.
  4. 설치가 끝나면 PowerPoint를 재시작합니다.
  5. 홈 탭 끝에 TEAMPPT 그룹 버튼이 보이면 성공입니다.

⚠ 관리자 권한은 최초 설치 1회만 필요합니다 (COM 등록). 이후 업데이트엔 필요 없습니다.

2 API 키 넣기 (최초 1회, 필수)

  1. 관리자에게서 받은 api-keys.json 파일을 준비합니다 (보안상 다운로드에 포함되지 않음).
  2. 탐색기 주소창에 %LOCALAPPDATA%\TeampptAddin\app\Assets 를 붙여넣어 폴더를 엽니다.
  3. 그 폴더에 api-keys.json 을 복사합니다.
  4. PowerPoint를 재시작하면 AI 기능이 켜집니다.

🔒 키는 비공개로 전달받으세요. 업데이트해도 넣어둔 키 파일은 그대로 유지됩니다.

3 제거하려면

  1. 같은 폴더의 uninstall.bat을 관리자 권한으로 실행합니다.
