Download & Install

설치하고 시작하기.

최초 1회만 설치하면,
이후 업데이트는 자동입니다.

최신 버전 다운로드 현재 버전 v1.0.0
설치 방법
1

설치하기 — 관리자 권한, 최초 1회만

  1. 받은 zip 파일을 압축 해제합니다. 저장 위치는 아무 데나 OK.
  2. 압축 해제한 폴더에서 install.bat을 찾아 마우스 우클릭 → 관리자 권한으로 실행합니다.
  3. 까만 창이 뜨고 "설치 완료"가 표시되면 창을 닫습니다.
  4. PowerPoint를 완전히 껐다가 다시 켭니다.
  5. 상단 홈 탭 맨 끝에 TEAMPPT 버튼이 보이면 성공입니다. 🎉
⚠️ 관리자 권한은 COM 프로그램 등록 때문에 이 최초 설치 1회만 필요합니다. 이후 업데이트에는 필요 없습니다.
2

API 키 넣기 — 최초 1회, 필수

AI 기능을 쓰려면 키 파일이 있어야 합니다. 관리자에게 따로 받으세요.

  1. 관리자에게서 api-keys.json 파일을 비공개로 받습니다.
    (보안상 다운로드 zip에는 포함되지 않습니다)
  2. 윈도우 탐색기 상단 주소창을 클릭하고, 아래를 그대로 붙여넣은 뒤 Enter를 누릅니다.
    %LOCALAPPDATA%\TeampptAddin\app\Assets
  3. 열린 폴더 안에 받은 api-keys.json 파일을 복사해 넣습니다.
  4. PowerPoint를 재시작하면 AI 기능이 켜집니다.
🔒 키 파일은 내 PC 안에만 저장됩니다. 업데이트를 해도 직접 넣어둔 키는 그대로 유지됩니다.
✕

제거하려면

  1. 설치할 때 사용했던 폴더에서 uninstall.bat을 찾아 관리자 권한으로 실행합니다.
자동 업데이트

한 번만 설치하면, 그 다음부터는 자동입니다.

자동 업데이트

다시 다운로드할 필요도, 관리자 권한도 필요 없습니다.
관리자(개발자)가 새 버전을 올리기만 하면, 모든 팀원에게 아래 과정이 자동으로 진행됩니다.

1
PowerPoint를 켭니다

TEAMPPT 애드인이 시작되면서 조용히 "새 버전이 있나?" 확인합니다. 내가 하는 작업에는 영향 없습니다.

2
새 버전 발견 → 백그라운드에서 다운로드

업데이트가 있으면 알아서 미리 받아둡니다. 하던 작업 그대로 계속하면 됩니다.

3
패널 상단에 배너가 나타납니다
✨ 업데이트 준비됨 — 지금 적용
4
버튼을 누르면 PowerPoint가 한 번 재시작됩니다

미루고 싶으면 안 눌러도 됩니다. 강제 적용은 없습니다.

✓
최신 버전으로 자동 교체 완료

api-keys.json을 비롯한 개인 설정은 모두 그대로 유지됩니다.
